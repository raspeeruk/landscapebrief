# OrgBrief — Design Brief

## Aesthetic Direction
**Institutional Editorial** — the McKinsey annual report of org chart tools. Cool, structured, quietly authoritative. Not warm-cozy editorial — this is boardroom-grade publishing. Every page looks like it belongs in a £50K deliverable.

## Target Audience
Executive recruiters and search firm researchers. They sell £50K+ reports to CHROs and CEOs who see McKinsey decks daily. They want to look as premium as they charge.

## Typography
- **Headings**: EB Garamond (400, 600, 800) — institutional serif
- **Body**: Public Sans (300, 400, 600) — government-grade clarity
- **Data/Mono**: Source Code Pro (400, 500) — precision for confidence scores and data
- **Scale**: 16px body / 52px h1 / 72px hero — 3x+ jumps

## Layout
- Max width 1200px for app content, full-bleed for chart viewer
- 64px+ between sections, 32px card padding
- Cards: 6px radius, thin stone border, gentle shadow
- Clean horizontal rules, no wave dividers or gradients

## Unique Hook
Confidence notation system — every inferred relationship gets a teal/amber/red dot with a percentage. Hover reveals reasoning.
